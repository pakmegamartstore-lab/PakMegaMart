## üé® REDESIGN STRATEGY

### Color Palette (Luxury Leather Theme)
```
Primary: #2D1F0F (Deep Chocolate/Leather Black)
Secondary: #8B6F47 (Warm Caramel)
Accent: #D4AF37 (Matte Gold - Limited Use)
Neutral: #F5F3F0 (Off-White/Cream)
Dark: #1A1410 (Near Black)
Accent 2: #C41E3A (Burgundy - For Discounts)
```

### Typography System
```
Display: Playfair Display (Serif) - Headlines, luxury feel
Body: Inter (Sans-serif) - Legibility, modern
Accent: Crimson Text (Serif) - Special highlights

Font Sizes:
H1: 3.5rem (Hero)
H2: 2.25rem (Sections)
H3: 1.5rem (Card titles)
Body: 1rem (16px)
Small: 0.875rem (14px)
```

### Component Improvements
1. **Buttons** - Enhanced shadows, smoother transitions, better affordance
2. **Cards** - Refined shadows, better image treatment, clearer hierarchy
3. **Badges** - New discount/quality badges with better visibility
4. **Navigation** - Sticky, enhanced styling, better hover states
5. **Modals** - Improved backdrop, smoother animations
